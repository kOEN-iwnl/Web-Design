Important Tags of HTML

<html> This tag is what helps the browser know that the document is a HTML document. Without this tag the browser will not know the code is a HTML document, so then it won't display what you have coded.
<head> The Head tag is a part of a HTML document. While content is not type here, this space is used for linking CSS files as well as loading custom fonts. This tag creates a section on the document that allows the coder to customize the aesthetics of the page.
<body> The body tag is used to tell the browser to read what is coded below. The body is used for creating the content and buttons on a webpage.

How to Add a Link to Your Webpage

Now your average web page will contain some sort of link to redirect you to a different site or place. And now you want to attach your own link in your own document. How? It's a real simple process. By using the <a> tag, you defined your own link. When you see clickable links on a webpage, it also says the website name but not the full url. But how do you do that on your own? Well while using the <a> tag, you need to also end with a </a>tag. So your full command line should look like <a href='URL HERE'>WHAT THE LINK SAYS ON A WEBPAGE</a>.
